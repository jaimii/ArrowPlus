## ArrowPlus

ArrowPlus is a modular Minecraft plugin that allows for customizeable arrow velocities. Designed for Project: Kompass.
This plugin hooks and has a API depedency on [ProtocolLib.](https://github.com/dmulloy2/ProtocolLib/)

| Command | Permission | Usage                                           |
| -------- | -------- |-------------------------------------------------|
| /arrowplus reload | arrowplus.reload | Reloads the plugin configuration. (default: op) |
